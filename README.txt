Instrucciones para compilar el APK:

1. Instala Cordova: npm install -g cordova
2. Desplázate a esta carpeta: cd mi_vida_cordova_full
3. Añade plataforma Android: cordova platform add android
4. Coloca tu icono en www/assets/icon.png, tu fondo en night_sky.jpg, tu audio en DannyOcean-BEBE.mp3
5. Ejecuta: cordova build android --release
6. El APK se generará en platforms/android/app/build/outputs/apk/release/app-release.apk
